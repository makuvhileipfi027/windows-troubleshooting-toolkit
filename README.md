# Windows Troubleshooting Toolkit (PowerShell)

## Project Overview

The Windows Troubleshooting Toolkit is a PowerShell script designed to help diagnose common Windows system issues.
It performs automated checks such as disk space analysis, system information retrieval, running services inspection, and internet connectivity testing.



## Features

* Disk space check
* Running services inspection
* Internet connectivity testing
* System information retrieval



## Technologies Used

* PowerShell
* Windows Command Line
* Windows Management Instrumentation (WMI)


## How to Run the Script

1. Download the repository from GitHub.
2. Open Windows PowerShell.
3. Navigate to the folder containing the script.

Example:

cd Documents

4. Run the script:

.\troubleshoot.ps1
