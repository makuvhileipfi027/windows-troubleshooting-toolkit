# Windows Troubleshooting Toolkit

## Description

This project demonstrates a PowerShell troubleshooting script used to collect system information and diagnose common Windows issues.

## Features

* Check disk space
* List running services
* Test internet connectivity
* Display system information

## Tools Used

* Windows 10
* PowerShell

## How to Run

1. Open PowerShell
2. Navigate to the script folder
3. Run the script:

.\troubleshoot.ps1
